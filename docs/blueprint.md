# **App Name**: Examplify

## Core Features:

- Role-Based Access: Admin and Candidate role-based access and dashboards.
- Exam Creation: Exam creation with multiple-choice questions, including options and correct answers.
- Exam Timer: Exam timer for candidates with automatic submission upon expiration.
- Results Review: Candidate exam result review after submission.
- User Authentication: Secure candidate registration and login using email/password or Google.

## Style Guidelines:

- Primary color: Moderate blue (#5D9CEC) for trust and focus.
- Background color: Very light blue (#E9F2FA), close to white, creating a clean and calm testing environment.
- Accent color: Soft violet (#A685E2), to bring attention to important actions or notifications, complementing the analogous palette.
- Body and headline font: 'Inter' sans-serif for clear and modern readability.
- Use clear and professional icons for navigation and actions.
- Responsive design for optimal viewing on various devices.
- Subtle animations and transitions to enhance user experience without distraction.
